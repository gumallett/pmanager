Overview
====

A project to manage and search a video inventory.

Prerequisites
====

 * JDK 11  
 * Docker
 * Yarn

Building
====

Uses gradle,

```bash
gradle openApiGenerate
gradle build
```

UI
====

```bash
cd ui
yarn install
yarn start
```
